Sulfur' Keyboard
==============
Android Gingerbread Keyboard with Myanmar(Zawgyi-one) Language

![SulfurKeyboard Screenshot][1]

- Simple Gingerbread Keyboard.
- Zawgyi auto correction.
- Suggestion for Myanmar Language.
- And more....
